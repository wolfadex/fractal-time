# Fractal Timeline

---

## Quick Start

- Install [yarn](https://yarnpkg.com/en/)
- Run `yarn global add firebase-tools`
- Run `firebase login`
- Run `yarn`
- Run `yarn dev`
- In a separate terminal run `yarn start`
